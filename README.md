📌 Overview

The Career Guidance System is a machine learning-based web application designed to help students choose the most suitable career path. Unlike traditional systems that rely only on academic scores or aptitude tests, this system analyzes a combination of academic performance, aptitude test results, and behavioral traits to provide accurate and personalized career recommendations.

The system uses the Random Forest Algorithm to improve prediction accuracy and ensure reliable results.

🚀 Features
📊 Takes academic scores as input
🧠 Conducts aptitude test
🧍 Includes behavioral/personality analysis
🤖 Uses Machine Learning for prediction
🎯 Provides personalized career recommendations
🌐 User-friendly web interface
⚡ Fast and efficient predictions
